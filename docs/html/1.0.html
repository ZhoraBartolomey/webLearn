HTML - основная структура

HTML имеет свою структуру. Корень является тег html. Внутри него находятся 2 тега: head и body.

Элемент head

head предназначен для передачи браузеру необходимой мета-информации. Например, заголовок сайта, описание сайта, ключевые слова и связи с CSS.

Тег meta: 

<meta name="название" content="значение">

Тег title: 

<title>Название сайта</title>

Мы вернёмся к head чуть попозже, когда будем изучать CSS.

Элемент body

Body - это элемент, в котором должны находится теги, которые будут отображены пользователю.

Общая структура документа

<html>
    <head>
        <title>Мой сайт</title>
        <meta charset="utf-8"/>
    </head>
    <body>
        <p>
            Мой сайт
        </p>
    </body>
</html>
