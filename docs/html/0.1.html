HTML - Введение

HTML - это язык разметки. Он нужен для создания каркаса сайта.

Теги

HTML код состоит из элементов - тегов. Теги можно вкладывать в теги и делать иерархическую структуру.

Тег открывается знаком "меньше" затем идёт название, атрибуты, затем знак "больше". Примерно вот так: <p>.

Также у тегов есть атрибуты. Это набор пар ключ-значение. Они пишутся в HTML вот так: class="my-class".

Затем идёт текст, другие теги и затем тег закрывается почти так же как и открывался: </p>

Вот что получилось:

<p class="my-class">
   Текст
   <span>Тоже текст</span>
</p>
