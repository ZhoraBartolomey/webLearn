Web Learn - это интерактивный онлайн учебник, который поможет вам в обучении веб-разработки

С чего начнём?

HTML

HTML (Hyper-Text Markup Language) - это язык разметки, каркас сайта.

Рекомендуется изучать его первым.

CSS

CSS (Cascading Style Sheets) - язык таблиц стилей, который позволяет прикреплять стили к HTML.

JS

Javascript - это язык программирования, который был создан для добавления динамики веб-страницам.
