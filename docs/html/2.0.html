Параграф <p>

Для того, чтобы писать текст в HTML, есть такой тег <p>.

В него нужно всего лишь написать текст:

<p>Ваш текст</p>

Стилистика текста <b>, <i>, <u> и <s>

<b>

Для того, чтобы сделать текст жирным, есть тег <b> или <strong>:

<b>Жирный</b> текст

Результат: жирный текст

<i>

Для того, чтобы сделать текст курсивом есть тег <i> или <em>:

<i>Курсивный</i> текст

Результат: курсивный текст.

<u>

Подчеркивать текст можно тегом u:

<u>Подчеркнутый текст</u>

<s>

Выделять Зачеркивать текст можно тегом <s>:

<s>Хороший</s> Плохой текст

Результат: Хороший Плохой текст
